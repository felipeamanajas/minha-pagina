Sobre mim

Criei esta página para aplicação das habilidades adquiridas ao longo do curso de Desenvolvedor Front-End na Arbyte.

Meu primeiro contato com informática foi no meu primeiro trabalho, atendente em uma lan-house aos 15 anos.
Aos 17 anos me isncrevi no curso de Sistema de informação, o qual cursei até o 3º semestre, mas tive que trancar por conta do trabalho.
Com 19 anos passei no vestibular para Engenharia Civil na Universidade de Mogi das Cruzes e mudei para São Paulo.
Já aqui em São Paulo, para sustentar os estudos e o custo de vida, trabalhei como vendedor nextel, SAC, estagiei e trabalhei na engenharia e consegui me formar no final de 2017.
Ao longo de 4 anos trabalhando na construção civil, me decepcionei muito com ambiente de trabalho e em fevereiro de 2020 resolvi deixá-lo e buscar algo novo.
Comecei a procurar em todo canto, algo que realmente me encatasse, e quando eu vi o anúncio da Arbyte no Instagram, eu soube que essa era minha oportunidade de migrar de carreira.

Tenho como principais metas para 2020:

  1. Melhorar autoestima e confiança
  2. Realizar a migração da área da construção civil para área de Tecnologia
  3. Sair do sedentarismo e ter uma vida mais saudável
  4. Reservar mais tempo para as pessoas importantes na minha vida

html5

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

CSS 3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets, pela qual se define estilos para um projeto web. Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

Git e GitHub

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

GitHub é uma plataforma de hospedagem de código-fonte com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.
